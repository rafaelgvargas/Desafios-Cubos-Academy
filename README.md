# Desafios

Bem-vindo ao repositório de desafios da Cubos Academy! Este repositório foi criado para armazenar e gerenciar os desafios propostos durante o curso, tanto para back-end quanto para front-end.


## Sobre

Este repositório contém soluções e exemplos de código para os desafios fornecidos pela Cubos Academy. O objetivo é proporcionar um local centralizado para praticar e revisar os desafios.
